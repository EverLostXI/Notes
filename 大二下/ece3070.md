## System Verilog

HDL（硬件描述语言）是为了搭建大规模的运算电路而发明的语言。

System Verilog是Verilog的超集，类似于C与C++的关系。SV中优化了Verilog中的一些痛点，并且加入了强大的模拟功能

1. 基础框架

   module代码块用来定义你的元件

   ```verilog
   module example (
       // 在开头的括号中，可以定义这个元件的输入和输出
       input logic a, b,
       output logic f
   );
       // 还可以创建元件的内部信号并为它赋“值”，类似变量
       logic na;
       assign na = ~a;
       
       // 可以实例化逻辑门，并且把你定义的信号接入其中。第一位是输出，后面可以有不定位数的输入
       and g0 (f, na, b);
       
   endmodule
   ```

2. RT设计中常用

   ```verilog
   // RT设计中需要多位宽的数据传输，可以在定义元件输入输出的时候设置
   module example (
       // 可以用parameter语法设置参数，方便修改位宽
       #(parameter W = 3)
       // 然后直接给输入输出信号设置位宽
       input logic [W-1: 0] D,
       output logic [W-1:0] Q
   );
   ```

## 组合逻辑设计

1. 香农展开：对于任意逻辑表达式，可以递归使用香农展开进行分解
   $$
   F(x_1,x_2,\dots,x_n)=x_i\cdot F(x_i=1)+x_i\cdot F(x_i=0)
   $$
   在上式中，$F(x_i=1)$与$F(x_i=0)$还可以分别使用香农展开进行分解

   * 这样做的意义是，可以把复杂的表达式用嵌套MUX进行表示



## RT级设计

工业级设计的最小单元不再是逻辑门，而是已经搭好，经过检验的基础元件。

* 硬件领域的RT设计规范，等同于软件领域的封装。

FSM-D架构（又称硬件线程）的设计，将电路分为两个主要部分：

1. 数据通路（Datapath）：用来处理数据的部分。由基础元件组成，如算术单元、寄存器
2. FSM：通过发送指令，控制数据通路的部分。是一个有限状态机。

* 数据通路中每条线可以包含多位数据
* 一个RT过程（寄存器$\rightarrow$组合逻辑$\rightarrow$寄存器）应该在一个时钟刻内完成
